# javascript
## string methods
trim - removes the spaces from the begining and end and returns a new string does not change the old string.
```
str.trim();
```

### strings are immutable.
whenever you make a change in a string then the original string stays the same but javascript create a new sting with modifications.

## toLowerCase() and toUpperCase()
these two methods transforms the string to lower or upper case.
```
str.toLowerCase();
str.toUpperCase();
```
